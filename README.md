# Utföra *blink* med Arduino IDE och Pulsivo 

Detta repo hanldar om Arduino IDE och Plusivo och hur man kan utför ett "blink test"

## Förutsättningar 
För att kunna utföra detta behöver du ett Pulsivo kit och programmet Arduino IDE nerladda på din dator, dock kommer förklaring och bilder av program och kitetet nedan för förtydligande

### Arduino IDE

Arduino IDE är ett gratis *open source* mjukvaroprogramm för att progrmmera och ladda upp kod för fysika kort, de vill säga att fyiska programeringar man gör kan laddas upp
och styras via mjukvaroprgrammet Arduino IDE. 

Detta är hur progrmmet

### Pulsivo 

Pulsivo är ett märke som skapar kit med elekroniska komponenter, som passar bra för nybörjare som lär sig om *micocontrollers* och mikroprocesser.
